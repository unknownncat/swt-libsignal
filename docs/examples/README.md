# Runtime examples

Scripts de validação real usados pela suíte de testes:

- `crypto-runtime-check.ts`
- `session-runtime-check.ts`
- `group-runtime-check.ts`
- `repository-runtime-check.ts`
- `storage-runtime-check.ts`

## Executar manualmente

```bash
npx tsx docs/examples/run-all.ts
```

Explicação: executa todos os checks e imprime um resumo JSON com os resultados.
